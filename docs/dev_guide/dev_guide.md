# ContentForge Development Guide

This document is intended to provide a guide for developers who are interested in contributing to 
the ContentForge project. 

It will cover the following topics:
- [Versioning](./versioning.md) - About the versioning name rules.
- [Change History](./change_history.md) - About the change history.

## Code Style
We use the [PEP 8](https://www.python.org/dev/peps/pep-0008/) style guide for Python code. 
Please make sure to follow it when contributing to the project.
